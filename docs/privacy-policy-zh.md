# 隐私政策

最后更新日期：2025-11-09

欢迎使用"ABB 程序阅读器"（"本应用"）。本应用由个人开发者 Jack 提供和运营。我们重视您的隐私与数据安全，并依据适用法律（包括适用的欧盟 GDPR 等）保护您的个人信息。本隐私政策说明我们如何收集、使用、存储与分享信息，以及您享有的权利。

## 我们收集的信息
当前版本不主动收集以下个人或设备数据，除非您在应用内明确触发相关功能（目前未集成任何第三方 SDK）：
- 设备信息（型号、系统版本、设备标识符等）— 不收集
- 崩溃 / 错误日志 — 暂未集成专用崩溃分析 SDK，系统级崩溃日志不被自动上传
- 使用统计数据（功能使用频率等）— 不收集
- 位置信息 — 不收集
- 广告标识符（Android Advertising ID 等）— 不收集
- 其他敏感数据（账号、通讯录、精确位置等）— 不收集

如未来版本引入相关功能，我们将在更新政策或应用内提示后再收集。届时您可选择是否授权。

## 用途与共享
由于当前不收集上述数据，本应用不存在对个人数据的常规处理或共享。
若未来为改进稳定性与体验需要采集匿名统计或崩溃信息，将仅用于：
- 性能与稳定性改进
- 功能质量评估
- 问题诊断与修复

除法律法规要求或为实现必要技术托管（例如提供本隐私政策的静态网页服务）外，我们不会出售或向第三方提供可识别您身份的个人信息。

## 第三方服务 / SDK
当前版本：无。
若未来接入（如 Firebase Crashlytics / Analytics 等），我们将：
- 在隐私政策中列出具体名称与隐私政策链接；
- 在应用内更新必要授权说明或同意流程。

## 权限使用
当前应用仅在需要文件访问时请求相关系统权限（例如选择并读取 ABB 机器人程序文件）。不涉及位置、摄像头、麦克风等高敏感权限。权限仅用于对应功能，不会在后台滥用。

## 未成年人保护
本应用面向技术/工业场景，不专门针对未成年人。若未成年人使用，应在监护人指导与同意下进行。

## 数据保留与安全
目前不存储或保留个人数据。若后续引入数据采集，我们将遵循最小化与必要性原则，并在政策中明确：
- 崩溃日志：示例 90 天；
- 汇总分析数据：示例 12 个月；
- 访问控制与安全措施。

## 您的权利
如未来采集个人数据，您可根据适用法律行使访问、更正、删除、撤回同意等权利。您可通过邮箱或应用内设置发起请求。目前由于不收集个人数据，上述请求暂不适用。

## 国际合规（GDPR 等）
由于面向"全球（含欧盟）"用户，如未来产生个人数据处理，我们将：
- 明确处理的合法基础（如用户同意、合法利益等）；
- 支持数据主体权利（访问、删除、可携带、限制处理、反对等）；
- 必要时说明跨境传输与保护措施。
